# Structural Design Pattern

Structural design pattern focuss on how organizing classes and objects (Geek for Geeks). __This pattern managing how objects assembly and become larger structure__ (Refactoring.guru). Imagine a poker card, card is an object and we made a pyramid from those card, how we structuring the card is structural design pattern. There are a few structural design patterns :
- Adapter
- Bridge
- Composite
- Decorator
- Faccade 
- FlyWeight
